# Runse — 需求澄清与提示词工程

将模糊意图转化为结构化需求文档，反推润色提示词，针对小白用户推荐技术方案。

## 安装

在支持 Skill 的 Agent 里直接说：

> 帮我安装这个 skill：https://github.com/wllqxls/runse_skill

## 触发

- `/runse`
- "帮我整理下需求"
- "我想做个东西但不知道从哪下手"
- "帮我分析下这个想法"

## 做了什么

分四轮推进：
1. **挖真实意图** — 追问 6 个关键问题，把一句话想法变成清晰需求
2. **拆可执行需求** — 按模块拆分详细开发要求 + 输出润色后的提示词
3. **推荐技术方案** — 小白友好对比表 + 强烈推荐 + 环境准备说明
4. **收尾** — 输出完整需求文档，告诉用户下一步

## 适合

- 不懂代码的小白，想开发东西不知道从哪下手
- 脑子里有想法但说不清楚
- 想让 AI 帮忙分析该用什么技术
